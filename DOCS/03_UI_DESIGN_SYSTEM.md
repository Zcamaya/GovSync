# UI Design System

GovSync uses a consistent, desktop-oriented PySide6 visual language that emphasizes clarity, spacing, and controlled interactions.

## Visual principles

- Keep the interface readable and structured.
- Preserve consistent spacing around cards, dialogs, and tables.
- Favor rounded containers and restrained visual decoration.
- Make important actions obvious without overloading the layout.

## Shared conventions

- Reuse shared styling constants where available.
- Keep form layouts predictable and aligned.
- Avoid hidden state in the UI when switching accounts or modules.
- Ensure panels remain usable at common desktop sizes.

## Component guidance

- Tables: keep content visible, use clear clipping and spacing.
- Dialogs: maintain a clear modal relationship and avoid overflow.
- Navigation: keep labels aligned with actual destinations.
- Panels: use consistent section headers, spacing, and action placement.

## Accessibility and responsiveness

- Preserve meaningful labels and object names.
- Keep focus order predictable.
- Avoid layout assumptions that make panels unusable in narrower windows.
